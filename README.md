# Number Guessing Game

A simple browser-based number guessing game built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies.

## How it works

The game picks a random number between 1 and 100. Enter a guess and it'll tell you if you're too high or too low, and keep count of your attempts, until you find the number.

## Run it locally

No build step needed. Just clone the repo and open `index.html` in a browser:

```bash
git clone https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
cd YOUR-REPO-NAME
open index.html
```

(On Windows, double-click `index.html` instead of using `open`.)

## Play it online

If GitHub Pages is enabled for this repo, it'll be live at:

```
https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
```
